The Sports Bank Widget is an aggregation of the web's best sports content. Adding this widget to your website will allow you to display the web's best football and baseball content on the internet!


== Description ==

Imagine all of the Best Sports sites from around the internet aggregated into one place, a widget you can easily embed into your website or as a Wordpress Plugin. Well you can stop wishing, The Sports Bank has put together an aggregation of the Webs Best Sports Content (NBA NFL NHL MLB ETC) ready to easily display on your website. Adding this widget to your website will allow you to display the web's best football and baseball content on the internet! Do you own a great sports website? Request that we add your headlines here --> http://www.thesportsbank.net/nba/add-the-new-sports-bank-stories-widget-plug-in-get-your-feed-added/


== Installation ==


1. Upload `fantasysports.zip` to the `/wp-content/plugins/` directory via plugin plugin section of your wordpress install
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar

== Frequently Asked Questions ==

= How do I add my website to the Sports Network? =

Send request to thesportsbank@gmail.com Only established websites with quality sports content are considered for inclusion to this sports network. Only those webmasters which run this widget on their website will have their content displayed on the widget. If you remove the widget from your website your website will be programmatically removed from the network. Basically, if you run the widget on your website then your website will be included in the network syndication and if you ever remove the widget you’ll be removed from the network.
